# Simple Project Template

This is a simple template for small projects presentation.

Built in VueJS@3 + Vite@4.

[Demo](https://63ccd57cc2d9a54bfb1deb90--tubular-klepon-1041c6.netlify.app)

## Usage

Install dependencies:

```shell
npm install
```

Run locally:

```shell
npm run dev
```

Build:

```shell
npm run build
```
